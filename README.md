Interpreter
===========
***BETA***
A simple boolean expression interpreter

The syntax is as follows:				
E::=B						
B::=LT					
T::=^LT					
 ::=vLT 	          
 ::=+LT                       
 ::=-LT                      
 ::=		                			
L::=A					         
 ::=~A					
A::=t				
 ::=true				
 ::=f					
 ::=false				
 ::=(B)	         			
 ::=cons
